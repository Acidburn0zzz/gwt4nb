Quick Start Guide to GWT4NB

  1. Make sure you are using a compatible version of NetBeans. We recommend using NetBeans 5.5.x for working with the current version of GWT4NB. The same binary works also with NetBeans 5.0 and development builds of NetBeans 6.0, but there are known compatibility issues.

  2. Download and install the Google Web Toolkit
  3. Install GWT4NB using the NetBeans Update Center Beta:
    • Choose Tools > Update Center from the main menu to open the Update Center wizard.
    • Select the "NetBeans Update Center Beta"

    • Add GWT4NB in the "Select Modules to Install" section:

    • Complete the remaining pages of the wizard to install the module.

    Alternatively the plugin can be installed manually
  4. See a flash-based tutorial on creating a simple application using GWT4NB.
  5. Refer to the GWT Developer Guide for instructions on using GWT. More information on how GWT4NB integrates GWT Toolkit with the NetBeans build infrastructure can be found in the GWT4NB under the hood document
See also: Creating GWT RPC Services Tutorial

Existing Web Applications can be GWT-enabled by adding the Google Web Toolkit framework in Project Properties-Frameworks
